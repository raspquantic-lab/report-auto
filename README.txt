준공서류 자동 생성기 — 코드 생성형 9종 v3.4

원본 XLSX를 포함하거나 실행 중 불러오지 않습니다.
index.html, style.css, app.js, workbook-schema.js 코드만으로 9개 시트를 생성합니다.

생성 서류
- 준공검사조서
- 준공검사 결과보고서
- 용역감독관 준공 감독조서
- 입회관 의견서
- 용역 완결 보고서
- 안전 보건 이행 확약서
- 안전 보건 협의체 운영내용
- 안전교육 및 작업장 점검 결과
- 위험성 평가표

GitHub Pages 사용
압축을 풀고 4개 웹 파일을 저장소 최상위에 업로드하세요. README.txt는 선택사항입니다.

주의
- 공사인부는 현재 양식의 표 범위에 따라 최대 17명까지 입력할 수 있습니다.
- Excel 생성에는 ExcelJS CDN을 사용하므로 최초 실행 시 인터넷 연결이 필요합니다.
- 위험성 평가표의 평일 계산은 토요일과 일요일만 제외하며 공휴일은 별도로 제외하지 않습니다.

v3.2 변경사항
- 위험성 평가표 시트 추가 및 개별 선택 생성 지원
- 대대장 이름 입력란 추가: 위험성 평가표 M7에 반영
- 시공자 이름을 J7 및 N12:N14에 반영
- 공사명을 B8 및 A12에 반영
- 평가일(M8)은 실착공일 기준 평일 3일 전으로 자동 계산
- 개선 예정일(L12:M14)은 실착공일 기준 평일 2일 전으로 자동 계산
- 위험성 평가표 날짜는 YY-MM-DD 형식으로 출력
- 위험성 평가표 인쇄영역은 A1:N14로 설정


v3.3 변경사항
- 위험성 평가표 행 높이와 열 너비를 최신 원본 양식에 맞춤
- 위험성 평가표 인쇄영역 A1:N14 유지
- A8:N14 전체 안쪽·바깥쪽 테두리를 얇은 실선으로 통일
- H7 왼쪽 테두리 추가


v3.4 변경사항
- 위험성 평가표 A12 공사명 셀에 자동 줄 바꿈 적용

v3.5 화면 개선사항
- 생성할 서류를 00번으로 변경하고 첫 번째 카테고리로 이동
- 모든 대분류를 기본 접힘 상태의 펼침형 카드로 변경
- 대대장 이름을 담당자 카테고리로 이동
- 받는이를 공사 정보 카테고리로 이동
- 입회관 입력란을 별도 선택 입력 카테고리로 분리
- 공사 정보 안에 계약·정산 금액, 주요 일정, 입찰·기간 변경 소분류 추가
