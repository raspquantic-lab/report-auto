준공서류 자동 생성기 — 코드 생성형 8종 v3.0

원본 XLSX를 포함하거나 실행 중 불러오지 않습니다.
index.html, style.css, app.js, workbook-schema.js 코드만으로 8개 시트를 생성합니다.

생성 서류
- 준공검사조서
- 준공검사 결과보고서
- 용역감독관 준공 감독조서
- 입회관 의견서
- 용역 완결 보고서
- 안전 보건 이행 확약서
- 안전 보건 협의체 운영내용
- 안전교육 및 작업장 점검 결과

GitHub Pages 사용
압축을 풀고 4개 웹 파일을 저장소 최상위에 업로드하세요. README.txt는 선택사항입니다.

v3.0 변경사항
- 안전·보건 관련 3개 시트 추가 및 개별 선택 생성 지원
- 회의장소 입력란 추가(기본값: 기술관리실 회의실)
- 교육내용 입력란 추가(기본값: 안전 교육 및 안전위해요소 확인 등)
- 공사인부 직급/이름을 한 명씩 추가·삭제하는 입력 목록 추가
- 공사인부의 순번, 시공사명, 직급, 이름을 두 안전·보건 표에 공통 반영
- 안전교육 및 작업장 점검 결과 제목을 올바르게 수정
- 교육 교관을 공사감독관 계급과 이름 사이 공백 한 칸으로 표시
- 기존 공사 정보와 담당자 입력값을 새 시트에 재사용

주의
- 공사인부는 현재 양식의 표 범위에 따라 최대 17명까지 입력할 수 있습니다.
- Excel 생성에는 ExcelJS CDN을 사용하므로 최초 실행 시 인터넷 연결이 필요합니다.
